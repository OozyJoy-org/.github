# [OozyJoy.org](https://www.oozyjoy.org)

OozyJoy.org is a community-driven platform dedicated to sharing joyful and uplifting content. Our mission is to spread positivity and happiness through stories, art, and experiences that inspire and bring joy to people's lives.

## Projects

- **OozyJoy Manifesto**: The principles that guide the board game experience.
- **OozyJoy Bedtop Game**: A fun and engaging sexual board game designed to bring people together and create orgasmic moments.
- **OozyJoy Mobile App**: An upcoming mobile application that will allow users to access OozyJoy content on tablets.
